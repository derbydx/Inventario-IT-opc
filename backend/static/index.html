Importar / Exportar Datos

Notas importantes para importar Activos

Columnas requeridas: AssetTag, Descripcion, Marca, Modelo, Serie, Categoria

Columnas opcionales: Sitio, AsignadoA

AsignadoA debe ser el email de la persona (no el nombre ni la cedula). Si el email existe, el activo se importa con status Checkout. Si esta vacio, queda como Available.

Los sitios y personas deben existir en el sistema antes de importar activos que los referencien.

El archivo debe ser .xlsx (Excel). La importacion es todo o nada: si una fila falla, no se importa nada.

Descarga una plantilla en la pestaña "Plantillas" para ver el formato exacto.

Exporta los datos del sistema a un archivo Excel.
